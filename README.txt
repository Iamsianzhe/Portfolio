微鲤海外效果视觉设计实习生作品集网站框架

使用方法：
1. 解压 zip 文件。
2. 双击 index.html 即可在浏览器打开。
3. 修改 index.html 里的文字、邮箱、微信、作品名称。
4. 将真实作品视频、图片放入项目文件夹后，把占位区域替换为 <video> 或 <img>。
5. 后续可以部署到 GitHub Pages、Netlify、Vercel，生成一个可投递链接。

文件说明：
- index.html：网页结构
- styles.css：Apple 风格视觉、毛玻璃、动效、响应式布局
- script.js：滚动显现、作品弹窗、移动端菜单、卡片动效
